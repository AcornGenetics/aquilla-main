How the instrument works

Starting a Run

The Run screen is where you start a test after loading samples.

Profile

Select a profile that defines temperature steps, cycles, and timing.

Run Name

Enter a descriptive name. If unchanged, the system auto-assigns one (e.g. Run 28).

Drawer

Use Open Drawer and Close Drawer to insert or remove tubes. The drawer must be closed before a run can begin.

Results

  • Detected (red/orange) — target is present in sample.
  • Inconclusive (yellow) — signal present but borderline.
  • Undetected (green) — no target is present in sample.
  • Deactivated (grey) — channel not used by this profile.

If a Result Is Inconclusive

An inconclusive result means the amplification curve could not be confidently classified as either Detected or Undetected. This may occur due to weak amplification, irregular curve behavior, low signal strength, or borderline threshold crossing. When that happens:

  1. Review the amplification curves in history for irregular behavior or weak amplification.
  2. Refer to the analysis help documentation for additional interpretation guidance.
  3. If the result is still inconclusive, repeat the test.

History

The History screen lists all completed runs so you can review results later.

Run Name

The name assigned to the run at the time it was started.

Date & Time

When the run completed.

Profile

The profile used for that run.

Result

A summary of detected, inconclusive, or not detected tubes.

Delete

Removes selected runs from history permanently.

Profiles

Profiles define how the instrument performs a test — temperature steps, cycle counts, and timing.

Creating a Profile

  1. Tap New Profile.
  2. Enter a name.
  3. Add steps (temperature + duration).
  4. Tap Save.

Saved profiles appear in the Run screen dropdown.

Bundled Profiles

Pre-configured profiles are available by default and cannot be deleted.

Run Detail

The Run Detail screen provides a deeper view of an individual run.

Metadata

Displays the run name, date, profile, and result summary.

Amplification Graph

Shows FAM and ROX fluorescence curves across PCR cycles.

Metrics

  • Detected — tubes with a confirmed signal.
  • Inconclusive — tubes with borderline results.
  • QC Status — Pass if no inconclusive tubes, otherwise Review.

Wi-Fi Setup

Connect the instrument to a network from the Settings screen.

Connecting to a Network

  1. Tap Settings in the top navigation, then open the Wi-Fi & System tab.
  2. Tap Scan for networks and wait a few seconds for the list to appear.
  3. Tap your network. If it is secured, enter the password (use Show to check it).
  4. Tap Connect. A green status dot confirms the instrument is online.

Managing Saved Networks

Previously joined networks appear under Saved Connections. Tap Forget to remove a network you no longer use.

Software Updates

Updates are applied only when you choose to. Nothing is downloaded or restarted automatically.

Checking for Updates

When an update is available, a badge appears on the Settings button. Open Settings and select the Updates tab to see the current status.

Applying an Update

  1. In Settings → Updates, tap Update Now to apply an available update, or Later to dismiss the reminder.
  2. While the update runs, leave the device powered on — do not power it off until it reports Update complete.
