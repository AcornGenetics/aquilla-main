How the instrument works

Starting a Run

The Run screen is where you start a test after loading samples.

Profile

Select a profile that defines temperature steps, cycles, and timing.

Run Name

Enter a descriptive name. If unchanged, the system auto-assigns one (e.g. Run 28).

Drawer

Use Open Drawer and Close Drawer to insert or remove tubes. The drawer must be closed before a run can begin.

Results

  • Detected (red/orange) — target signal successfully amplified.
  • Inconclusive (yellow) — signal present but borderline.
  • Not Detected (green) — no amplification observed.

History

The History screen lists all completed runs so you can review results later.

Run Name

The name assigned to the run at the time it was started.

Date & Time

When the run completed.

Profile

The profile used for that run.

Result

A summary of detected, inconclusive, or not detected tubes.

Delete

Removes selected runs from history permanently.

Profiles

Profiles define how the instrument performs a test — temperature steps, cycle counts, and timing.

Creating a Profile

  1. Tap New Profile.
  2. Enter a name.
  3. Add steps (temperature + duration).
  4. Tap Save.

Saved profiles appear in the Run screen dropdown.

Bundled Profiles

Pre-configured profiles are available by default and cannot be deleted.

Run Detail

The Run Detail screen provides a deeper view of an individual run.

Metadata

Displays the run name, date, profile, and result summary.

Amplification Graph

Shows FAM and ROX fluorescence curves across PCR cycles.

Metrics

  • Detected — tubes with a confirmed signal.
  • Inconclusive — tubes with borderline results.
  • QC Status — Pass if no inconclusive tubes, otherwise Review.
