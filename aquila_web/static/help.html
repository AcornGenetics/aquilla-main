Run History Profiles Run Detail

Run

The Run screen is where you start a test after loading your samples.

Profile

Select a profile that defines the temperature steps, number of cycles, and timing for the test.

Run Name

Enter a descriptive name for the run. If left unchanged, the system automatically assigns a name (e.g., Run 1, Run 2).

Drawer Controls

Use Open Drawer and Close Drawer to safely insert or remove samples. The drawer must be closed before a run can begin.

Run Button

Starts the test using the selected profile. Once a run begins, the Run button is locked until the test completes.

Results Overview

After a run finishes, results are summarized using clear status indicators:

  • Detected (green): Target signal was successfully amplified.
  • Inconclusive (yellow): A signal was present but unstable or borderline.
  • Not Detected: No amplification was observed.

History

The History screen lists completed runs so you can review results later.

Run Name

The name assigned to the run.

Date

The date and time the run completed.

Settings

The profile used for the run.

Result

A summary of detected, inconclusive, or not detected tubes.

View

Opens the detailed run view with amplification curves.

Delete

Removes selected runs from history. Deleting a run permanently removes it from the device.

Profiles

Profiles define how the instrument performs a test.

Name

A recognizable label for the profile (e.g., Acorn PCR).

Volume

Sample volume used for the test (uL).

Stages & Steps

Each stage includes a temperature and duration, defining how the instrument cycles during the run.

Creating a Profile

  1. Click New Profile.
  2. Enter a name and sample volume.
  3. Add stages and steps (temperature + time).
  4. Click Save to store the profile.

Saved profiles appear in the Run screen for selection.

Run Detail

The Run Detail screen provides a deeper view of an individual run.

Metadata

Displays the run name, date, profile used, and result summary.

Amplification Graph

Shows FAM and ROX fluorescence curves across PCR cycles.

Why This Matters

The shape and consistency of the curves help indicate whether results are valid, noisy, or inconclusive.
